Table --- Addresses

  • AddressID -- Int32 (key)
        Contains the unique identifier on each row of the Addresses table
  • DBCommand -- DBCommandEnum (int)
        Identify the database command for local database sync
  • AddressTVItemID -- Int32
        Link to the TVItems table with the unique identifier
  • AddressType -- AddressTypeEnum (int)
        Address type
  • CountryTVItemID -- Int32
        Link to the TVItems table representing the country
  • ProvinceTVItemID -- Int32
        Link to the TVItems table representing the province
  • MunicipalityTVItemID -- Int32
        Link to the TVItems table representing the municipality
  • StreetName -- String (max=200)
        Street name of the address
  • StreetNumber -- String (max=50)
        Street number of the address
  • StreetType -- StreetTypeEnum (int)
        Street type of the address
  • PostalCode -- String (min=6) (max=11)
        Postal code of the address
  • GoogleAddressText -- String (min=10) (max=200)
        Google address provided by Google Location Service
  • LastUpdateDate_UTC -- DateTime
        Contains the date of the last update in UTC
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
