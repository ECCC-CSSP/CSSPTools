Table --- DocTemplates

  • DocTemplateID -- Int32 (key)
        Contains the unique identifier on each row of the DocTemplates table
  • DBCommand -- DBCommandEnum (int)
        Identify the database command for local database sync
  • Language -- LanguageEnum (int)
        Document template language
  • TVType -- TVTypeEnum (int)
        Tree view item type
  • TVFileTVItemID -- Int32
        Link to the TVFiles table with the unique identifier
  • FileName -- String (max=150)
        File name of the document template
  • LastUpdateDate_UTC -- DateTime
        Contains the date of the last update in UTC
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
