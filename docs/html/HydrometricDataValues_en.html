Table --- HydrometricDataValues

  • HydrometricDataValueID -- Int32 (key)
        Contains the unique identifier on each row of the HydrometricDataValues table
  • DBCommand -- DBCommandEnum (int)
        Identify the database command for local database sync
  • HydrometricSiteID -- Int32
        HydrometricSite link to parent HydrometricSites table item
  • DateTime_Local -- DateTime
        Date and time of observation. Almost all data is coming from the water office web site.
  • Keep -- Boolean
        Keep the data within the database. Sometime the database might get some data from the water office web site as temporary data which can be removed at a later date
  • StorageDataType -- StorageDataTypeEnum (int)
        Storage data type can be archived, forcasted or observed
  • HasBeenRead -- Boolean
        The stored data has been read from the weather office web site
  • Discharge_m3_s -- Double (min=0) (max=100000)
        Discharges in cubic meters per second
  • DischargeEntered_m3_s -- Double (min=0) (max=100000)
        Discharges entered in cubic meters per second
  • Level_m -- Double (min=0) (max=10000)
        Levels in meters
  • HourlyValues -- String
        String containing hourly values
  • LastUpdateDate_UTC -- DateTime
        Contains the date of the last update in UTC
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
