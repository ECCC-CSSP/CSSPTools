Table --- HydrometricDataValues

  • HydrometricDataValueID -- Int32 (key)
        Contient l'identifiant unique sur chaque ligne de la table HydrometricDataValues
  • DBCommand -- DBCommandEnum (int)
        Identifie la command de base de données pour syncroniser avec la base de données local
  • HydrometricSiteID -- Int32
        Lien au parent CydrometricSite à l'item de la table HydrometricSites
  • DateTime_Local -- DateTime
        Date et temps de l'observation. Pratiquement tous les données proviennent de site web water office
  • Keep -- Boolean
        Conservez les données dans la base de données. Parfois, la base de données peut obtenir des données du site Web du bureau hydrométrique en tant que données temporaires pouvant être supprimées ultérieurement.
  • StorageDataType -- StorageDataTypeEnum (int)
        Type de données sauvegardez peut être archivés, prévisions ou observés
  • HasBeenRead -- Boolean
        Les données stockées ont été lues sur le site Web du bureau météorologique
  • Discharge_m3_s -- Double (min=0) (max=100000)
        Débits en mètres cube par second
  • DischargeEntered_m3_s -- Double (min=0) (max=100000)
        Débits entrés en mètres cube par second
  • Level_m -- Double (min=0) (max=10000)
        Niveaux en meters
  • HourlyValues -- String
        Texte contenant les valeurs horaires
  • LastUpdateDate_UTC -- DateTime
        Contient la date de la dernière mise à jour en (UTC)
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID qui a fait le dernier changement
