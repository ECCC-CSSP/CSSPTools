  • DBCommand -- DBCommandEnum (int)
        Identify the database command for local database sync
  • PolSourceSiteTVItemID -- Int32
        Link to the TVItems table representing pollution source site

  • Site -- Int32 (min=0) (max=1000)

  • SiteID -- Int32 (min=0) (max=1000)

  • IsPointSource -- Boolean

  • InactiveReason -- PolSourceInactiveReasonEnum (int)
        Pollution source inactive reason

  • LastUpdateDate_UTC -- DateTime
        Contains the date of the last update in UTC
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
