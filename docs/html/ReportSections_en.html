Table --- ReportSections

  • ReportSectionID -- Int32 (key)
        Contains the unique identifier on each row of the ReportSections table
  • DBCommand -- DBCommandEnum (int)
        Identify the database command for local database sync
  • ReportTypeID -- Int32
        Link to the ReportTypes table representing the report type
  • TVItemID -- Int32
        Link to the TVItems table representing the item
  • Language -- LanguageEnum (int)
        Language of item
  • Ordinal -- Int32 (min=0) (max=1000)
        Ordinal
  • IsStatic -- Boolean
        Is static --- the text will not change in time
  • ParentReportSectionID -- Int32
        Parent report section ID
  • Year -- Int32 (min=1979) (max=2050)
        Year for which the text is applicable
  • Locked -- Boolean
        Locked --- not currently used
  • TemplateReportSectionID -- Int32
        Template report section ID --- not currently used
  • ReportSectionName -- String (max=100)
        Report section name
  • ReportSectionText -- String (max=10000)
        Report section text
  • LastUpdateDate_UTC -- DateTime
        Contains the date of the last update in UTC
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
