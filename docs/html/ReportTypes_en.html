Table --- ReportTypes

  • ReportTypeID -- Int32 (key)
        Contains the unique identifier on each row of the ReportTypes table
  • DBCommand -- DBCommandEnum (int)
        Identify the database command for local database sync
  • TVType -- TVTypeEnum (int)
        TV type
  • FileType -- FileTypeEnum (int)
        File type
  • UniqueCode -- String (max=100)
        Unique code --- used in code for knowing which document and type of document to create
  • Language -- LanguageEnum (int)
        Language of item
  • Name -- String (max=100)
        Name of the report type
  • Description -- String (max=1000)
        Description of the report type
  • StartOfFileName -- String (max=100)
        Start of file name
  • LastUpdateDate_UTC -- DateTime
        Contains the date of the last update in UTC
  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
