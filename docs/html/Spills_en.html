        Link to the TVItems table representing the municipality
  • InfrastructureTVItemID -- Int32

  • EndDateTime_Local -- DateTime

  • AverageFlow_m3_day -- Double (min=0) (max=1000000)

  • LastUpdateDate_UTC -- DateTime

  • LastUpdateContactTVItemID -- Int32
        Contact TVItemID who made the last changes
