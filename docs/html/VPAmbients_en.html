Table --- VPAmbients

  • VPAmbientID -- Int32 (key)
        Contains the unique identifier on each row of the VPAmbients table

        Identify the database command for local database sync
  • VPScenarioID -- Int32
        Link to the VPScenarios table representing the visual plumes scenario

        Background concentration in most probable number per 100 milli-Litres
  • PollutantDecayRate_per_day -- Double (min=0) (max=100)
        Pollution decay rate per day
  • FarFieldCurrentSpeed_m_s -- Double (min=0) (max=10)

  • FarFieldCurrentDirection_deg -- Double (min=-180) (max=180)

  • FarFieldDiffusionCoefficient -- Double (min=0) (max=1)
